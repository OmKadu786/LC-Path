LCPath Privacy Policy

Last updated: June 11, 2026

LCPath is an educational productivity tool designed to help users prepare for coding interviews. We take your privacy seriously and are committed to transparently explaining how we handle your data.

1. Information We Collect

When you use the LCPath extension, we access the following data strictly to provide the core AI service:

  • Website Content: We read the LeetCode problem description, your active code editor content, and submission results to provide context-aware AI hints.
  • Personal Communications: The chat messages you type and send to the AI coach.

2. How We Use Your Information

The code you write and the chat messages you send are forwarded to the DeepSeek API solely for the purpose of generating personalized hints and guidance. We do NOT use this data for marketing, profiling, or any other unrelated purpose.

3. Data Storage and Transfer

We do NOT store your chat history or LeetCode progress on our remote backend servers. All user data is stored locally on your own device via Chrome's local storage.

When you request a hint, your code and chat prompts are transmitted securely via HTTPS to our proxy server and then to the DeepSeek API. They are processed in real-time and immediately discarded by our backend. No permanent logs of your code are kept on our servers.

4. Third-Party Services

We use the DeepSeek API to generate AI responses. DeepSeek's handling of data is governed by their respective privacy policy.

5. Contact Us

If you have any questions about this Privacy Policy, please contact the developer directly via the Support tab on the Chrome Web Store listing.
